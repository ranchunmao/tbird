# profile

## zshell

- add softlink .zshrc.local under the home directory to point the zshrc.local file
- edit '.zshrc' to source the personal profile
```
source ~/.zshrc.local
source ~/.zshrc.secret
```

# notes

- .zshrc.secret has a copy in the evernote
